# Xenon Shell - pretty much just like Powershell, but smarter. 

Useful AI agent. Xenon Shell can do anything regular command prompt can, but except that it can also:
  * Read, analyse and edit your code
  * Create, edit and delete directories and files

## Works on
  * **HTML/CSS** - UI
  * **Vue** - UX
  * **Node.js** - backend
  * **pollinations.ai** - text generation API

## Features
  + Completely free
  + Works online - no supercomputer required

## How to use
 * **Option 1**: Install and run
   + Full functional
 * **Option 2**: Open index.html in your browser
   + Only text generation

## How to install
  * Traditional way
    1. Go to https://xenomorphed.usite.pro/xenon-shell.html
    2. Click "Download"
    3. Done
   
  * Old School way
    1. Clone the repository
       ```
       git clone https://github.com/xenomorphed/xenon-shell.git
       ```
    2. Check if node.js is installed
       ```
       node -v
       npm -v
       ```
    3. Build
       ```
       npm install electron --save-dev
       npm install electron-builder
       npm run build
       ```
    4. Run the installer from ./dist
    5. Done
